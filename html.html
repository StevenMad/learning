Les Tags HTML

Le langage HTML est un langage balisé

Pour écrire du code html il faut procéder comme suit:

la première balise à écrire est celle-ci:

<html>

A l'interieur le code est divisé en differentes parties

<head></head> : Le head est l'en-tete

Cette en-tete sert à déclarer des données qui ne sont pas affichées directement sur la page

Par exemple la balise <meta> qui permet de déclarer les meta données

Exemple : <meta charset="utf-8">

Cette balise déclare que la norme utilisée pour l'encodage des caractères est utf-8

<body></body> : C'est dans cette balise que se trouvent les contenu à afficher sur la page

La page HTML se ferme avec la balise </html>

Structure de la balise

Il y a deux types de balises:

  • Les balises normales : <div>Du contenu</div>

    Ce sont les plus utilisées.

    Elles commencent par un < et suit le type de la balise

    Exemple de types: div, p (paragraph), a(lien)

    Entre la balise ouvrante et la balise fermante se trouve du contenu. Ce contenu est le texte affiché sur l'ecran

  • Les balises auto-fermantes : <input/>

    Ces balises sont utilisées lorsqu'il n y a pas de contenu à l'intérieur d'une balise

    Elles commencent par un < et suit le type de la balise

    Exemple de types: div, p (paragraph), a(lien)

Les balises peuvent contenir des attributs. Ces attributs peuvent être placé dans un ordre quelconque.

Un attribut est structuré de la manière suivante:

nom-de-l-attribut = "valeur de l'attribut"

Les attributs permettent de preciser des comportements ou des informations à la balise.

Prenons l'exemple suivant:

<input type="text" name="username" placeholder="Inserez le nom" class="inputClass"/>

La balise est une balise appelée input

C'est une balise auto-fermante

Elle contient plusieurs attributs:

un attribut type qui indique le type de l'input

un attribut name qui indique le nom de l'input. Il sera important pour récupérer les données d'un formulaire

un attribut placeholder qui indique le texte à afficher quand l'input est vide

un attribut class qui indique le nom de la classe à appliquer à l'input
