Javascript Basics

Ecrire du javascript

1+1 calcule la somme 1+1

console.log("hello world"); affiche dans la console la phrase hello world

var x=1 déclare une variable nommée x à qui on assigne la valeur 1

; Marque la fin d'une ligne d'instruction

{ ... } "{" designe le début d'un bloc et "}" désigne la fin du bloc

var x=1;

x=x+1;

La premiere instruction assigne 1 à x. La seconde instruction incrémente la valeur de x de 1

Les boucles et les conditions

for(init ; cond a remplir ; iteration){
...
}

init : Désigne l'état initial de la boucle

cond à remplir: tant que la condition est remplie lancer la suite d'instructions

iteration : à la fin des instructions appliquer l'itération

... : suite d'instructions

while(cond à remplir){ ... }

Tant que la condition est remplie effectuer la suite d'intructions comprise dans le bloc

Les fonctions

Declaration d'une fonction

function nomDeLaFonction(attr1,attr2){
...
}

function: mot indispensable pour signifier que l'on déclare une fonction

nomDeLaFonction: le nom qu'on veut donner à la fonction

attr1, attr2: une suite de paramètres à assigner à la fonction. On peut avoir 0,1,ou plusieurs paramètres

Les paramètres sont espacés par des ,


Retour d'une fonction

Les fonctions sont utilisées pour executer une suite d'instructions qui pourraient être utilisées plusieurs fois. En général les fonctions retournent un résultat

Dire à une fonction qu'elle renvoie un résultat s'écrit comme suit : return ... ;

Appel d'une fonction

Considérant que la fonction nomDeLaFonction a été déclarée, on l'appelle ensuite comme ceci:

nomDeLaFonction(monParam1,monParam2);
